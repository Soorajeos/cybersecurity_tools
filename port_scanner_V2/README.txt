port_vuln_scan_V2 is an vulnerablity scanner tool for port scanning and comparing the captured banners.

The accuracy of the scan depends on the timeouts. The timeouts can be changed by editing s.settimeout().

The txt file of the vulnerable softwares can be entered and the port_vuln_scan_V2 will compare it with grabbed banners.

clone the files and run port_vuln_scan_V2.py to run the tool.

Designed to run from the command line.
