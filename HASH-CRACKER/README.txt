hash_cracker.py is a simple tool that can used to perform dictonary attacks on password hashes.

This program is now made for cracking md5 and sha1 hashes, but it can be updated to crack any type of password hashes by adding a few lines of code.

The algorithm can be same, just need to customize a bit for cracking other hash types.

The customized password list can be easily generated with crunch tool.

NOTE : Designed to be executed from terminal.
