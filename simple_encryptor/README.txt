simple_encryptor is a script that can be used to encrypt files with symmetric encryption.

clone the script.

# Generate the key and store in some safe place.

python3 generate_key.py

enter the path to save the key


# Encrypt the files

python3 encrypt.py <key.key/path to key> <file/or path to file>

# Decrypt the file 

python3 decrypt.py <key.key/ path to key> <file/or path to file>
