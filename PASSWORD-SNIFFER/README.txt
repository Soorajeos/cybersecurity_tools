password_sniffer is a tool that can used for sniffing the packets and extracting the creentials out of it.

password_sniffer inspects the packets captured and extracts the username and passwords out of it.

This tool can be used with ARP-SPOOFER or any other tools used for arpspoofing.

NOTE : For extracting the credentials from the https websites, a combination of other tools should be used for downgrading the webpage.
       For example : sslstrip for downgrading the SSL. 
       Tools for downgrading the TLS can be also found with doing research.
       Designed to be executed from the terminal.
